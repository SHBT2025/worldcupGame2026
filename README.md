一、项目概述

这是一个**2026年世界杯竞猜游戏**的HTML单页应用，包含三大核心功能模块：

| 模块 | 页面ID | 功能 |
|------|--------|------|
| 📊 **赛况首页** | `pg-home` | 小组赛赛程表、积分榜、小组第三名排名、淘汰赛赛程 |
| 🎮 **模拟赛果** | `pg-sim` | 复制当前赛况，自由预测未开赛比赛，系统自动推算积分 |
| 🎰 **模拟投注** | `pg-bet` | 模拟赔率、下注、投注单、投注历史、盈亏统计 |


半成品 正在修改中。 为了方便看赛果 ，目前市面的软件 画面太好了 就是看比分不直观 费劲。 没想到AI那么容易做 自己先用着 半成品也行。
